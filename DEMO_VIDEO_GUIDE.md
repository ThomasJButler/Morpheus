# Morpheus - Demo Video Recording Guide

> Script and recording guide for creating a compelling demo video

## Video Specifications

- **Duration:** 5-7 minutes
- **Resolution:** 1920x1080 (1080p minimum)
- **Format:** MP4 (H.264)
- **Frame Rate:** 30fps or 60fps
- **Audio:** Clear narration (optional background music)

## Recording Tools

**Screen Recording:**
- macOS: QuickTime, OBS Studio
- Windows: OBS Studio, Camtasia
- Linux: OBS Studio, SimpleScreenRecorder

**Video Editing:**
- DaVinci Resolve (free)
- iMovie (macOS)
- Adobe Premiere Pro

```

Just like Morpheus revealed the truth about The Matrix,
this RAG chatbot reveals insights from your data."
```

```
"Traditional RAG systems have problems:
- They use only semantic similarity, missing keyword matches
- They can't adapt their search strategy
- They return context without reasoning

This results in poor retrieval and generic responses."

[Show side-by-side comparison: Basic RAG vs Morpheus]
```

```

1. Agentic Intelligence - Claude reasons about HOW to search

3. Query Enhancement - Automatic rewriting and expansion

The result? 48% better retrieval performance."

[Show performance benchmark chart]
```

## Recording Checklist

### Pre-Recording
- [ ] Clear browser cache
- [ ] Close unnecessary tabs/applications
- [ ] Set display to 1920x1080
- [ ] Prepare sample documents
- [ ] Test microphone levels
- [ ] Practice script 2-3 times

### During Recording
- [ ] Speak clearly and at moderate pace
- [ ] Pause between segments for editing
- [ ] Show cursor movements deliberately
- [ ] Keep mouse movements smooth
- [ ] Re-record any mistakes immediately

### Post-Production
- [ ] Cut long pauses
- [ ] Add text overlays for key points
- [ ] Add background music (subtle, non-distracting)
- [ ] Add timestamps/chapters
- [ ] Color correct for consistency
- [ ] Export at 1080p, 30fps minimum
